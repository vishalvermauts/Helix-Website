# Helix Ecosystem Website

This is the sleek, Vercel-ready promotional landing page for the **Helix Ecosystem**, including the **Helix Engine**, the **HelixFlow Gateway**, and the **Helix Brain Diagnostic Lab**. It is built using the industry-standard modern web stack to achieve an ultra-premium Dark Obsidian & Neon Minimalist aesthetic.

## Tech Stack
- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React

## Getting Started

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run the Development Server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deploying to Vercel

This repository is strictly optimized for [Vercel](https://vercel.com).
1. Push this codebase to a new GitHub repository (e.g. `Helix-Website`).
2. Log into your Vercel Dashboard.
3. Click **Add New Project** and import the GitHub repository.
4. Leave all build settings as default (`npm run build`).
5. Click **Deploy**. Your premium landing page will be live globally in seconds!
